# geofs-alarms

[Install GeoFS-Alarms](https://github.com/fengshuo2004/geofs-alarms/raw/master/geofs-alarms.user.js) | [Report a bug](https://github.com/fengshuo2004/geofs-alarms/issues) | [Talk on Discord](https://discord.gg/YAfH5Z4)
